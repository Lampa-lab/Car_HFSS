# Car_HFSS
Задание выполнялось согласно 5 варианту (f = 1.005 ГГц) в программном пакете Ansys Electronics Desktop Student 2025 R2 проводилось на стандартной модели автомобиля, которая состоит из 5 основных элементах: body (корпус); wheels, wheels_1, wheels_2, wheels_3 – колёса. Однако для корректного расчета рассеяния (ЭПР) был использован только элемент body, для этого все элементы колёс были отключены в окне дерева модели, а сам элемент body делаем идеальным проводником (значение Perfect E).

После подготовки компонентов модели к работе, следует добавить настройки анализа. Для этого в менеджере проекта задаем в analysis задаём частоту (для моего варианта 1005 МГц или 1,005 ГГц). 
 
Затем создаём множество падающих волн с помощью вкладки excitations в сферической системе координат, где указываем диапазоны в плоскостях фи и тета, шаг и ориентацию вектора E. Измерения проводились с двумя множествами:
1)	Множество падающих волн в системе координат фи от 0 до 360 градусов с шагом в 2 градуса с единичной волной в системе координат тета в позиции 90 градусов, единичный вектор E ориентирован вдоль плоскости тета.

2)	Множество падающих волн в системе координат тета от 0 до 180 градусов с шагом в 2 градуса с единичной волной в системе координат фи в позиции 90 градусов, единичный вектор E ориентирован вдоль плоскости фи.
 
После подготовки модели к расчету запускаем расчет модели в рамках пакета HFSS Design. Если все расчеты будут проведены корректно, то требуется вывести результат. Для этого заходим во вкладку result, чтобы построить диаграммы с помощью инструмента 3D Polar.
 
Для более наглядного понимания рассеивания электромагнитной волны на автомобиле наложим результат построения инструментом 3D Polar на основную модель, что даст диаграмму отражённой волны от автомобиля.
 
Также можно воссоздать визуализацию данного процесса с помощью построения поля E на объекте.
